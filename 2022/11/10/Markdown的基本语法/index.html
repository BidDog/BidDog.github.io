Markdown基本语法

发表于 2022-11-10 分类于 Markdown学习

介绍了使用Markdown做记录时会用到的基本语法。

标题语法

添加#，实现标题。

Markdown语言 预览效果
## 这是二级标题 这是二级标题
###### 这是六级标题 这是六级标题

注意：#后需要添加一个空格。
注意：共有6级标题。

换行语法

在行尾添加“结尾空格”（在一行的末尾添加一个或者多个空格），或添加<br>，实现换行。

Markdown语言 预览效果
第1行。
第2行。		 第1行。
第2行。
第1行。<br>
第2行。		 第1行。
第2行。

换段语法

添加1行空白行来开始下一段。

Markdown语言 预览效果
第1段第1行。
第1段第2行。

第2段第1行。
第2段第2行。		 第1段第1行。
第1段第2行。

第2段第1行。
第2段第2行。

强调语法

添加不同数量的*，可将文本设置为粗体或斜体。

Markdown语言 预览效果
**粗体**：你**好**啊 粗体：你好啊
*斜体*：你*好*啊 斜体：你好啊
***粗体+斜体***：你 ***好*** 啊 粗体+斜体：你 好 啊

注意：不建议添加标题元素，会造成目录中的标题混乱。

列表语法

分为有序列表和无序列表。

有序列表

使用数字+.+空格创建有序列表，使用Tab缩进创建子列表。

Markdown语言：

示例效果：

  1. 水果的营养
  2. 蔬菜的营养
    1. 胡萝卜的营养
    2. 大白菜的营养
    3. 西红柿的营养
    4. 西兰花的营养
  3. 肉类的营养

无序列表

使用-（或者*、+）+空格创建无序列表，使用Tab缩进创建子列表。

Markdown语言：

示例效果：

  • 拿起筷子
    • 眼看到筷子
    • 手拿起筷子
  • 夹菜
  • 把菜放进嘴里

注意：不要将-、*、+混合使用（原因可能是改变符号被判定为换段）。

列表中嵌套其他元素

保留列表连续性的同时，嵌套其他元素，该元素前需添加4个空格（相当于1个Tab），不同元素间需用空白行分开。

Markdown语言示例：

示例效果：

  1. 做饭，吃饭

  2. 洗碗，刷筷

    不做饭的人去洗碗，天经地义。————哥布迅

  3. 把碗放到柜子里

  4. 休息

上例为列表嵌套块引用的方法，嵌套段落、图片、列表、围栏代码块时方法同理。
